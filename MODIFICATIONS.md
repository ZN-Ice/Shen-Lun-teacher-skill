# 申论教学工具修改说明

## 修改日期
2026-03-02

## 修改目标
改进申论教学工具，确保材料和问题正确关联，并在推送题目时一起显示。

## 修改内容

### 1. 新增 `scrape-materials.js`
改进的爬虫脚本，能够：
- 从网页提取申论试卷的给定资料（材料1、材料2、材料3...）
- 提取试卷中的所有题目（第一题、第二题、第三题...）
- 分析题目引用的材料（例如"根据给定资料3"）
- 为每个题目创建独立记录，并关联它引用的材料
- 如果题目未明确引用材料，则关联所有材料

**使用方法：**
```bash
cd /home/admin/.openclaw/workspace/Shen-Lun-teacher-skill
node scrape-materials.js
```

### 2. 修改 `send-question.js`
改进的题目推送脚本，能够：
- 清晰显示题目要求
- 显示题目关联的所有给定资料
- 改进材料显示格式，添加材料编号标识
- 优化答题提示

**使用方法：**
```bash
cd /home/admin/.openclaw/workspace/Shen-Lun-teacher-skill
node send-question.js
```

### 3. 新增 `check-materials.js`
材料关联检查脚本，用于验证题目和材料的关联情况。

**使用方法：**

## 数据结构说明

### 题目对象结构
```json
{
  "id": "q_hn_2024_sszj_3",
  "year": "2024年",
  "region": "湖南省考",
  "level": "省市卷",
  "title": "第三题",
  "question": "市妇联刘主席代表N市...",
  "requirements": "紧扣资料，内容全面...",
  "score": 30,
  "difficulty": "medium",
  "tags": ["儿童友好", "发言稿", "湖南"],
  "source": "爱真题",
  "url": "https://www.aipta.com/article/9596.html",
  "materials": {
    "3": "给定资料3的内容..."
  }
}
```

### materials 字段说明
- **类型**：对象（Object）
- **键**：材料编号（字符串，如 "1", "2", "3"）
- **值**：该材料的完整内容
- **关系**：一个题目可以关联一个或多个材料

## 工作流程

### 1. 爬取新题目
```bash
node scrape-materials.js
```
- 从爱真题网站爬取湖南省考申论真题
- 解析试卷结构，提取材料和问题
- 为每个题目关联对应的材料
- 保存到 `src/data/questions.json`

### 2. 检查材料关联
```bash
node check-materials.js
```
- 查看所有题目的材料关联情况
- 统计包含材料的题目数量
- 验证材料是否正确关联

### 3. 推送题目
```bash
node send-question.js
```
- 随机选择一道题目
- 显示题目要求
- 显示题目关联的所有给定资料
- 提供答题提示

## 待优化项

1. **材料内容清理**：某些材料内容中包含了重复的标题（如"材料3"），需要在爬取时清理
2. **材料引用识别**：需要更智能地识别题目引用的材料（处理多种表达方式）
3. **材料长度控制**：对于长材料，考虑截断或分页显示
4. **多材料关联**：优化题目引用多个材料时的显示格式

## 总结

通过这次修改，申论教学工具现在能够：
- ✅ 正确提取申论试卷的给定资料
- ✅ 正确提取试卷中的所有题目
- ✅ 建立题目和材料的正确关联
- ✅ 在推送题目时，同时显示对应的材料

现在用户可以在收到题目的同时，看到所有相关的给定资料，从而更好地理解题目背景并进行作答。
